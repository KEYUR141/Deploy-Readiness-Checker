# Repo Structure Check

1. REQUIRED files (BLOCKER if missing): README.md, .gitignore, lockfile
2. RECOMMENDED files (WARNING if missing): .env.example, Dockerfile
3. GOOD-TO-HAVE (INFO if missing): CONTRIBUTING.md, LICENSE, CHANGELOG.md
4. Report each as PRESENT or MISSING with severity.
